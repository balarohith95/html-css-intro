CSS----Cascading style sheets

It’s a styling language used to describe how HTML elements should look on a web page.
While HTML is used to structure content (like headings, paragraphs, and images),
CSS controls the presentation - such as colors, fonts, spacing, layouts, and animations.

RWD in CSS stands for Responsive Web Design

It’s a design approach that makes websites adapt to different screen sizes and
devices (like phones, tablets, and desktops) so that the content looks good and remains usable everywhere
